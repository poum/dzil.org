Dist::Zilla

Dist::Zilla est un programme pour rendre plus facile d'écrire, de packager, de gérer et de livrer des logiciels libres. Il cible les bibliothèques écrites avec le langage de programmation Perl et livrées sur CPAN. Si vous livrez des logiciels sur CPAN, Dist::Zilla peut vraisemblablement vous épargner pas mal de travail. Il possède des plugins pour automatiser des dizaines d'étapes fastidieuses comme:

  • ajouter les mentions de copyright et de licence
  • déterminer les pré-requis
  • construire les outils d'installation Makefile.PL ou Build.PL
  • taguer vos livraisons dans votre outil de gestion de code source
  • livrer sur CPAN
  • ajouter des tests pour la compilation, la documentation et le style
  • simplifier le processus de documentation
  • gérer les numéros de version
